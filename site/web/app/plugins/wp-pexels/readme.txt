== Description ==

This plugin help you search over million free photos from https://pexels.com then insert to content or set as featured image very quickly.

= Features =
- Search images with AJAX
- Quick insert image with original URL into content
- Upload image to Media Library then inserts into content (Pro version)
- Use image as featured image
- Multiple select support (NEW)

== Installation ==

1. Unzip the wp-pexels.zip
2. Copy wp-pexels folder to wp-content/plugins
3. Go to Plugins > Installed Plugins, find WP Pexels and click Active
4. Now when you create/edit the post, you can see the Pexels button about the content editor
5. Enjoy!
